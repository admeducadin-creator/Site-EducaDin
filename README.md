# 💰 EducaDin: Guia do Projeto | Project Guide

## 🇧🇷 Português

### 1. O que é o EducaDin?
O **EducaDin** é um site criado para ajudar jovens a cuidarem melhor do seu dinheiro. Ele foi desenvolvido como um projeto de estudo para o programa **Transforme-se**, focado em aprender o básico de como criar páginas para a internet.

### 2. Como o site funciona?

#### 🌓 Cores do Site
O site tem um botão (Sol e Lua) que permite escolher entre o modo claro ou escuro. O site "lembra" da sua escolha mesmo se você fechar a página.

#### 📰 Mural de Notícias
O site busca automaticamente notícias reais sobre finanças na internet para manter o usuário informado.

#### 🧮 Calculadora
Tem uma ferramenta que faz uma conta simples: ela mostra quantas horas ou dias você precisa trabalhar para conseguir comprar um produto.

### 3. Desafios
Como o site ainda é um protótipo, algumas partes podem ficar "bagunçadas" na tela do celular. Também precisamos organizar melhor o tamanho das letras no futuro.

### 4. Como baixar e testar 🚀

#### Passo 1: Baixar os arquivos
Você pode ter os arquivos no seu computador de dois jeitos:
* **Jeito Fácil (ZIP):** Clique no botão verde escrito **"Code"** lá no topo desta página e depois em **"Download ZIP"**. Depois, é só abrir a pasta no seu computador.
* **Jeito Git (Terminal):** Se você já usa o Git, digite no seu terminal:
    ```bash
    git clone [https://github.com/seu-usuario/educadin.git](https://github.com/seu-usuario/educadin.git)
    ```

#### Passo 2: Abrir o site
Você não precisa instalar nenhum programa especial. Basta seguir estes passos:
1. Entre na pasta que você baixou.
2. Procure pelo arquivo chamado `index.html`.
3. Clique duas vezes nele e o site vai abrir no seu navegador de internet (Chrome, Edge, Firefox, etc).

#### 🛠️ Ferramentas Usadas
- **HTML:** Para montar o "esqueleto" do site.
- **CSS:** Para as cores, fontes e visual.
- **JavaScript:** Para fazer as contas da calculadora e buscar as notícias na internet.

## 🇺🇸 English

### 1. What is EducaDin?
**EducaDin** is a website to help young people manage their money. It was created as a study project for the **Transforme-se** program, focusing on learning the basics of web development.

### 2. How the site works?

#### 🌓 Site Colors
The site has a button (Sun and Moon) to switch between light and dark modes. The site "remembers" your choice even if you close the page.

#### 📰 News Feed
The site automatically finds and shows real financial news from the internet.

#### 🧮 Calculator
There is a tool that shows how many hours or days of work you need to buy a specific product.

### 3. Challenges
Since the site is a prototype, some parts might look messy on mobile screens. We also need to organize the text sizes better in the future.

### 4. How to download and test 🚀

#### Step 1: Get the files
You can get the files on your computer in two ways:
* **Easy Way (ZIP):** Click the green **"Code"** button at the top of this page and then **"Download ZIP"**. Then, just open the folder on your computer.
* **Git Way (Terminal):** If you already use Git, type in your terminal:
    ```bash
    git clone [https://github.com/seu-usuario/educadin.git](https://github.com/seu-usuario/educadin.git)
    ```

#### Step 2: Open the site
You don't need to install any special software. Just follow these steps:
1. Open the folder you downloaded.
2. Find the file named `index.html`.
3. Double-click it and the site will open in your web browser (Chrome, Edge, Firefox, etc).

#### 🛠️ Tools Used
- **HTML:** To build the site's skeleton.
- **CSS:** For colors, fonts, and visual.
- **JavaScript:** For the calculator logic and to find news on the internet.
